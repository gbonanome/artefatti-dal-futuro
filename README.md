# Artefatti dal Futuro

11 scenari fittizi dal 2030: annunci, articoli, app, chat e pagine web da un futuro plausibile.

Progetto realizzato per il workshop **"Corner sui temi futuri dei CLab"**, evento celebrativo CLab — Ca' Foscari, 20 febbraio 2026.

**[Visita il sito](https://gbonanome.github.io/artefatti-dal-futuro/)**

## Cos'è

Una collezione di **oggetti digitali fittizi ambientati nel 2030** — interfacce, articoli, chat, email, profili social — pensati per rendere il futuro concreto e discutibile. Non prevedono il futuro: lo materializzano in artefatti navigabili, per stimolare discussione e immaginazione.
